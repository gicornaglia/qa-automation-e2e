# QA Automation E2E with Cypress

Projeto de automação de testes end-to-end utilizando Cypress, focado na validação dos principais fluxos de uma aplicação web: login, carrinho e checkout.

## Objetivo

Demonstrar uma estrutura de automação organizada e escalável, cobrindo cenários críticos do usuário final e aplicando boas práticas de QA.

## Fluxos automatizados

- Login de usuário
- Adição de produtos ao carrinho
- Processo de checkout

  e2e/
